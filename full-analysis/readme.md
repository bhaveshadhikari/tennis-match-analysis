# Full Analysis

This folder contains the results from processing a full 5-minute tennis video.

## Summary

- The entire 5-minute video was processed end-to-end.
- The output includes detected ball positions, combined detections, and shot classification results.
- Processed output can be viewed at the link below.

## Output Link
The full video can be viewed at: 
https://youtu.be/8-rpe6ig7ug

## Files

- `detections.csv`: detected rackets and person per frame 
- `ball_detections.csv`: detected ball coordinates per  frame.
- `combined_detections.csv`: merged detections and curated track data.
- `output_with_shot.csv`: final output with shot classification and hit event details.
